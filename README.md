# Hello Byte

Hello Byte is a real-time video streaming service similar to Google Meet, utilizing WebRTC and Socket.IO technologies.

## Tech Used:

- **ReactJS**
- **ExpressJS**
- **WebRTC**
- **Socket.IO**
